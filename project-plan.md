# CSCE 1015 GitHub Assignment 1 – Project Plan

## Goals
- Practice using Git branches and pull requests
- Use issues and a project board for basic project management
- Demonstrate individual contribution for the final grade

## Tasks
- Set up repository structure (src, docs, assets)
- Create and update README
- Create feature branches for changes
- Open pull requests and merge them
- Create issues and organize them on a project board

## Milestones
- v0.1 – Repo initialized and structure created
- v0.2 – Documentation added in docs/
- v0.3 – Example code or demo content in src/
